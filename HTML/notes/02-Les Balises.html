Les balises

Les balises blocs

L'ouverture et la fermeture d'une balise bloc sont séparés par le saut d'une ou plusieurs lignes créants ainsi un bloc de contenu. Différents blocs peuvent êtres imbriqués les uns dans les autres.
Un élément de type bloc occupe par défaut toute la largeur disponible.

Les balises lignes

Les éléments de type ligne sont ceux qui occupent un espace délimité par leur balise plutôt que d'intérompre le flux du contenu.
