Les Médias

Une page HTML peut afficher différents types de médias :

  • Images
  • Vidéos
  • Audios
  • Animations

Les Images

L'affichage d'une image se fait à l'aide de la balise Image, <img>, pour la doc suivez le lien !
Cette balise possède de nombreux attributs, 3 sont fortements recommandés pour un usage basique, il s'agit de :

  • <src>, pour Source
  • <alt>, pour Alternative
    • permet d'afficher un texte de remplacement si l'image ne charge pas
    • fournit une légende à l'image lorsqu'on passe la souri dessus
  • <title>, pour Titre, permet d'attribuer un titre à votre image et sert le référencement

Les vidéos

L'affichage d'une vidéo se fait à l'aide de la balise Video, <video>, pour la doc suivez le lien !
Seuls les 3 formats suivants sont reconnus par les navigateurs :

  • mp4
  • webm
  • ogg

Certains navigateurs ne peuvent lire qu'un seul des formats de video précédemment sité, il faut donc indiquer 3 fichiers sources à chaque fois que l'on veut intégrer une vidéo.
Il est souvent plus simple d'intégrer une vidéo via youtube et une balise <iframe>.
Youtube génère un code HTML qu'il est possible d'intégrer dans son propre code via la balise <iframe> (permet aussi d'intégrer des pops-up).
